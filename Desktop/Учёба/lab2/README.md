# Лабораторна робота №2  
## Створення HTML-форми, стилізація та валідація (JS)

### Мета роботи
- Навчитися створювати форми в HTML.
- Освоїти базову стилізацію форм за допомогою CSS.
- Реалізувати клієнтську валідацію введених даних на JavaScript.
- Отримати практичні навички роботи з Git та GitHub.

---

## Частина 1. HTML — створення форми

Створено файл **form.html**, який містить форму для реєстрації з такими полями:

- Ім’я
- Email
- Пароль
- Повтор пароля  
- Кнопка **«Надіслати»**

Форма містить елементи `<input>`, `<label>`, `<button>` та контейнери для виведення помилок.

---

## Частина 2. CSS — стилізація

У файлі **style.css** реалізовано:

- загальне оформлення форми (фон, тіні, шрифти),
- стилізацію полів введення,
- стилізацію кнопки з ефектами `:hover` та `:active`,
- ефекти `:focus`,
- стилізацію повідомлень про помилку та успішну відправку.

Стилі виконані через селектори класів та псевдокласи.

---

## Частина 3. JavaScript — валідація даних

У файлі **script.js** реалізовано:

- Перевірка, що поля «Ім’я» і «Повтор пароля» не порожні.
- Перевірка формату email через регулярний вираз.
- Перевірка довжини пароля (мінімум 6 символів).
- Перевірка збігу паролів.
- Виведення повідомлень про помилки під відповідними полями.
- Виведення повідомлення **«Форма успішно відправлена!»** у разі повної коректності даних.

Використано DOM API, регулярні вирази, методи роботи з класами та подія `submit`.

---

## Частина 4. Робота з Git

Виконано:

1. Створено гілку `lab2`.
2. Додано файли проекту командою:
git add form.html style.css script.js README.md .gitignore
3. Зроблено коміт:
git commit -m "feat: add form with validation"
4. Об’єднання гілки `lab2` з `main`:
git checkout main
git merge lab2
5. Відправлення репозиторію на GitHub.

---

## Файли проєкту

- `form.html` – HTML-форма.
- `style.css` – стилізація форми.
- `script.js` – JS-перевірка введених даних.
- `README.md` – опис лабораторної роботи.
